minhas Hard Skills

HTML (Linguagem de Marcação de Hipertexto)

É o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas.

CSS (Cascading Style Sheet)

Sua função principal é atribuir estilo a elementos escritos em HTML, ou seja, o CSS cuida da parte visual e estética de um site: cores, fontes, espaçamentos entre parágrafos, tabelas, variações de layouts, ajustes de imagem, tudo isso entra como estilização.

JavaScript

É uma linguagem de programação que adiciona interatividade ao seu site. Isso acontece em jogos, no comportamento das respostas quando botões são pressionados ou com entrada de dados em formulários; com estilo dinâmico; com animação, etc.

React

É uma biblioteca modular, o que significa que os componentes podem ser facilmente reutilizados e compartilhados entre diferentes partes da aplicação. Isso torna mais fácil manter e expandir uma aplicação à medida que ela cresce, e permite que equipes de desenvolvimento trabalhem de forma mais eficiente juntas.
